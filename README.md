# CipherRelay
Backend routing layer for a privacy-first end-to-end encrypted messaging system. Handles secure message delivery, session management, and device coordination without accessing user content.
A secure end-to-end encrypted messaging system.
---

## 🚀 How to Run

```bash
npm install
npm start
```
